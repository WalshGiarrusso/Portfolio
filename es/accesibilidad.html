Declaración de Accesibilidad

Garantizar la accesibilidad es la prioridad número uno de este sitio web.

Internet me ha permitido mejorarme toda mi vida. Yo nunca habría sido capaz de aprender las habilidades que llevaron a la creación de este sitio web sin internet. Yo creo que todas las personas deberían poder se benefican del internet como yo hice. La accessibilidad de la web es personal para mí, y yo lo tomo muy serio.

Al crear este sitio web, he considerado la accesibilidad mientras escribiendo cada línea de código. He hice un esfuerzo extremo proveer la misma experiencia para usarios de todo las capacidades mentales y físcos, además de usarios de todo medios technologicós. También, soy siempre trabajando mejorar en el trabajo para accesibilidad que ya hice.

Reclamación de Conformidad

Desde agosto de 2020, todas las páginas web de walshgiarrusso.com acatan completamente a WCAG 2.1 Nivel AAA. Este sitio de web depende de las siguientes tecnologías web para acatar a WCAG 2.1: HTML, CSS, JS, SVG, y PNG. Además de acatar de WCAG 2.1, este sitio de web también tratar seguir los últimos borradores de WCAG, y cambios de accesibilidad generales que no son exigidos por nada directrices.
