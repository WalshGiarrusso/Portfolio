Accessibility Statement

Ensuring accessibility is this website's number one priority.

The internet has enabled me to better myself my entire life. I never would have been able to learn the skills that lead to this website's creation without the internet. I believe everyone has a right to benefit from the internet just as I did. Web accessibility is personal for me, and I take it very seriously.

In creating this website, I have considered accessibility in writing every line of code. I have made an extreme effort to provide an equivalent experience for users of all physical and mental abilities, as well as all technological means. In addition, I am always working to improve the work towards accessibility that I have already done.

Conformance Claim

Since August 2020, all walshgiarrusso.com web pages fully conform to WCAG 2.1 Level AAA. This website relies on the following web technologies for conformance with WCAG 2.1: HTML, or Hypertext Markup Language; CSS, or Cascading Style Sheets; JavaScript, including jQuery and JSON, or JavaScript Object Notation; SVG, or Scalabe Vector Graphics; and PNG, or Portable Network Graphics. In addition to conforming to WCAG 2.1 Level AAA, this website also attempts to conform to the latest drafts of the Web Content Accessibility Guidelines, as well as general accessibility adjustments that are not mandated by any standards.
